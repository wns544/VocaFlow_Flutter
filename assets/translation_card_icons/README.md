# Translation Card Icon Assets

Source: `Redesign as Vector Art.zip`, provided by the project owner.

- `source_translation_cards.png`: original 1254 x 1254 image
- `translation_cards_vector.svg`: editable vector reconstruction
- `translation_cards_240.png`: large presentation size
- `translation_cards_80.png`: medium icon size
- `translation_cards_64.png`: dark-background sample size
- `translation_cards_52.png`: compact icon size
- `translation_cards_36.png`: small icon size

The original PNG is also used as `assets/app_icon.png`. Android launcher and
splash resources are generated from the same source at each density.
